奇异值和奇异值矩阵是什么

● 奇异值与奇异值分解 (SVD)

  奇异值 (Singular Values)

  奇异值是矩阵的固有属性，描述矩阵对向量进行"拉伸"的程度。

  对于一个 $m \times n$ 的矩阵 $A$，它的奇异值 $\sigma_i$ 是 $A^T A$（或 $A A^T$）的非零特征值的平方根：

  $$\sigma_i = \sqrt{\lambda_i(A^T A)}$$

  其中 $\lambda_i$ 是特征值。

  直观理解：如果把矩阵 $A$ 看作一个线性变换（把一个球映射为一个椭球），那么奇异值就是该椭球各半轴的长度。

  奇异值矩阵 (Singular Value Matrix)

  奇异值分解 (SVD) 把任意 $m \times n$ 矩阵 $A$ 分解为：

  $$A = U \Sigma V^T$$

  其中：

  ┌──────────┬────────────────┬───────────┬─────────────────────────────────────────────────────────────────────────┐
  │   矩阵   │      名称      │   大小    │                                  性质                                   │
  ├──────────┼────────────────┼───────────┼─────────────────────────────────────────────────────────────────────────┤
  │ $U$      │ 左奇异向量矩阵 │ $m \times │ 正交矩阵：$U U^T = I$，列是 $A A^T$ 的特征向量                          │
  │          │                │  m$       │                                                                         │
  ├──────────┼────────────────┼───────────┼─────────────────────────────────────────────────────────────────────────┤
  │ $\Sigma$ │ 奇异值矩阵     │ $m \times │ 对角矩阵（非负），对角线上是奇异值 $\sigma_1 \geq \sigma_2 \geq \dots   │
  │          │                │  n$       │ \geq \sigma_r > 0$，其余为 0                                            │
  ├──────────┼────────────────┼───────────┼─────────────────────────────────────────────────────────────────────────┤
  │ $V^T$    │ 右奇异向量矩阵 │ $n \times │ 正交矩阵：$V V^T = I$，列是 $A^T A$ 的特征向量                          │
  │          │                │  n$       │                                                                         │
  └──────────┴────────────────┴───────────┴─────────────────────────────────────────────────────────────────────────┘

  奇异值矩阵 $\Sigma$ 的结构

  对于一个 $m \times n$ 矩阵，$\Sigma$ 是一个"伪对角"矩阵：

  例如 $3 \times 4$：
  $$\Sigma = \begin{pmatrix}
  \sigma_1 & 0 & 0 & 0 \
  0 & \sigma_2 & 0 & 0 \
  0 & 0 & \sigma_3 & 0
  \end{pmatrix}$$

  例如 $4 \times 3$：
  $$\Sigma = \begin{pmatrix}
  \sigma_1 & 0 & 0 \
  0 & \sigma_2 & 0 \
  0 & 0 & \sigma_3 \
  0 & 0 & 0
  \end{pmatrix}$$

  关键性质

  1. 秩：非零奇异值的个数 = 矩阵的秩 $r$
  2. 范数：
    - 谱范数（2-范数）：$|A|_2 = \sigma_1$（最大奇异值）
    - 核范数（迹范数）：$|A|_* = \sum \sigma_i$
    - Frobenius 范数：$|A|_F = \sqrt{\sum \sigma_i^2}$
  3. 条件数：$\kappa(A) = \sigma_{\max} / \sigma_{\min}$（用于判断数值稳定性）
  4. 低秩近似（Eckart-Young 定理）：取前 $k$ 个最大奇异值，可以得到矩阵的最优秩-$k$ 近似

  应用场景

  - PCA（主成分分析）：对协方差矩阵做 SVD，奇异向量即为主成分方向
  - 数据压缩：扔掉小的奇异值，实现低秩近似，压缩图像/数据
  - 推荐系统：隐语义模型（如 Netflix 竞赛中的矩阵分解）
  - 伪逆矩阵：Moore-Penrose 伪逆 $A^+ = V \Sigma^+ U^T$
  - NLP 中的 LSA（潜在语义分析）
  - 图像去噪：保留大的奇异值，去掉噪声对应的小奇异值
